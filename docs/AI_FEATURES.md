# Sellog AI 기능 기획 및 로드맵

블로그·커머스 플랫폼에 **AI를 적극 활용**해 작성·SEO·운영을 고도화하기 위한 기획서.

---

## 1. 목표

- 글쓰기·상품 등록 시 **제목·설명·메타** 등을 AI로 제안해 작성 부담을 줄인다.
- **SEO·메타 설명** 자동 생성으로 검색 노출 품질을 높인다.
- (추후) **이미지 alt·카피 확장** 등으로 기능을 단계적으로 확장한다.

---

## 2. Phase 1 — 즉시 구현 (텍스트 생성)

| 기능 | 위치 | 설명 | 상태 |
|------|------|------|------|
| **제목 제안** | 글쓰기·글 수정 | 본문(또는 빈 값)을 바탕으로 글 제목 후보 1개 생성 | ✅ |
| **본문 요약(메타용)** | 글쓰기·글 수정 | 본문 요약 1~2문장 (메타 설명·SEO용) | ✅ |
| **상품명 제안** | 상품 추가·수정 | 키워드 또는 빈 값으로 상품명 후보 생성 | ✅ |
| **상품 설명 생성** | 상품 추가·수정 | 상품명·가격 등으로 짧은 상품 설명 생성 | ✅ |

**기술 요건**

- **API**: OpenAI API (`gpt-4o-mini` 또는 `gpt-3.5-turbo`) 사용. API 키는 `OPENAI_API_KEY` 환경 변수로 설정.
- **미설정 시**: 버튼은 노출하되, "API 키를 설정하면 AI 기능을 사용할 수 있어요" 안내.
- **보안**: 서버 액션에서만 API 호출. 키는 서버에만 두고 클라이언트에 노출하지 않음.

---

## 3. Phase 2 — 확장

| 기능 | 설명 | 상태 |
|------|------|------|
| **슬러그 제안** | 제목·본문 기반 URL 슬러그 후보 (글쓰기·글 수정) | ✅ |
| **이미지 alt 제안** | 썸네일(글쓰기/수정)·상품 이미지(상품 추가/수정) 대체 텍스트 | ✅ |
| **카드/소셜 카피** | SNS 공유용 짧은 문구 생성 (글쓰기·글 수정) | ✅ |
| **다국어 요약** | 본문 요약 영문 버전 (메타/SEO용, 글쓰기·글 수정) | ✅ |
| **AI 도우미 패널** | 글쓰기·글 수정 화면 접이식 "AI 도우미" (썸네일 alt · SNS 카피 · 영문 요약) | ✅ |

---

## 4. 환경 변수

```env
# .env.local (서버 전용, Git에 올리지 말 것)
OPENAI_API_KEY=sk-...
```

배포 시 Vercel 등에서 동일 변수 설정. 키가 없으면 AI 버튼은 동작하지 않고 안내만 표시.
