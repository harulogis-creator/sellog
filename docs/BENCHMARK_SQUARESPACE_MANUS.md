### 1.2 페이지별 디테일 (자녀 구역 학습)

- **website-design**
  - 섹션: Templates → Design Intelligence(Blueprint AI, Tailored to you, Edit drag‑drop, Preview styles, Explore layouts, Draft copy with AI) → Creative tools(반응형·미디어·모션·효과) → SEO & Analytics → Related(Domains, Email, Online Stores) → **How to design a website** 1~6 단계 → 24/7 Support → CTA.
  - **How to** 스텝이 명확한 번호+제목+한 문장.
- **create-a-blog**
  - 블로그 전용: 템플릿 → Features(Create/publish, Categorization, Control design, **Draft with AI**) → Marketing(Metrics, SEO·이메일) → Monetize(Email, Memberships, Online Store, Contacts) → **How to build your blog** 5단계 → FAQ 5개.
  - 블로그 수익화(멤버십·스토어·뉴스레터) 강조.
- **ecommerce-website**
  - 이커머스: 템플릿 → Sell products → **Squarespace AI**로 상품 설명·카피 → Manage(배송·결제·세금·주문) → 연동 로고 벽(Apple Pay, PayPal, Stripe, FedEx, USPS, UPS, Afterpay, Xero, Printful, TaxJar, Klarna 등) → Sell services.
  - 인용: "website design is the most important factor..." (Artips).
- **services**
  - 서비스 판매: Experiences / Consulting / Events 각각 4개 서브포인트, 하단 "More tools"(Marketing, Content & Memberships, Invoicing), **FAQ 5개** (서비스 판매·스케줄링·인보이스·결제).
- **invoicing**
  - 인보이스·클라이언트 관리: Elevated experience, Recurring, On-brand, Stay organized → Client management(Workflow, Intake forms, Toolkit, Dashboard) → 모바일(Invoicing, Pay Links, Tap to Pay) → **테스티모니얼 3건** → 관련 제품 3개 → FAQ 4개 → 24/7 Support.
- **solutions/creative-services**
  - 업종 랜딩: 한 문장 히어로 + 6개 페르소나 탭(Photographers, Graphic Designers, …) + 4개 카드(포트폴리오·계약/인보이스·SEO·결제) + 사회적 증거 + **고객 인용 3건** + FAQ + Blueprint AI + 24/7.
- **showcase**
  - "Made with Squarespace" — 한 줄 서브타이틀 + 스크롤 갤러리(포트폴리오·스토어·블로그·레스토랑·비즈니스), 실제 도메인 링크.

### 1.3 공통 패턴 (모든 자녀 구역)

- **상단**: 프로모 배너(20% 할인·도메인 $5) · 동일 메가메뉴(Products / Solutions / Resources) · Get started / Start a Free Trial.
