## 2. 작업 상세

### 2.1 고객 인용 섹션

| 항목 | 상세 |
|------|------|
| **위치** | 쇼케이스(Sellog로 만든 블로그) 직후, FAQ 직전 |
| **키** | `www.testimonial.title`, `quote1`, `author1`, `quote2`, `author2` (ko/en) |
| **UI** | 섹션 제목 + 2개 인용 카드(인용문 + 작성자), 심플 카드 또는 인용 스타일 |
| **콘텐츠** | 1차: 플레이스홀더/대표 문구. 추후 실제 고객 인용으로 메시지 값만 교체 |

### 2.2 고도화

- **접근성**: 인용 블록에 `role="region"`·`aria-labelledby`, 인용문은 `<blockquote>` 또는 `aria-label` 고려.
- **일관성**: 랜딩 톤(친근·전문)에 맞는 문구 길이 유지.

### 2.3 글로벌

- 인용 제목·문구·작성자 전부 `www.testimonial.*` 로 관리, 영문 전환 시 동일 키로 표시.
